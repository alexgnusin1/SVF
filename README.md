# SVF
Simple Verification Framework for Verilog &amp; SV Testbenches Development

SVF (Simple Verification Framework) is a free and Open Source Verilog-based framework for developing Verilog and SystemVerilog Testbenches. The framework introduces "verification patterns" required to quality Testbench development: 
- Verification components structuring
- Script examples for tests and regressions
- Untimed transactions communication concept
- Centralized logging and error collection
- The "end-of-test" procedure
- Error and Status reporting
- Functional Coverage collection for plain Verilog
- Functional Coverage viewer editing & merging capabilites

  SVF consists currently of the following elements:
  - Demo scripts folder
  - SVF support code
  - SVF Project examples
  - Functional Coverage code including coverage monitors and coverage viewer
  - Documentation
